# Portfolio Website with Admin Dashboard

A modern portfolio website designed for illustration commission services. This project includes an admin dashboard with CRUD functionality and an interactive chatbot to enhance client communication.

## 🚀 Features
- Responsive and modern UI design
- Admin dashboard for managing content
- CRUD functionality (Create, Read, Update, Delete)
- Interactive chatbot for user assistance
- Portfolio showcase for client works

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
